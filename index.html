Introduction

Dekka Dekka is a three-dimensional maze game with a two dimensional display. You play as an alien trying to get back to their home planet. You can navigate the maze with the arrow keys and rotate your view with on-screen buttons. I built it in Netbeans using Java Swing to design the user interface. This was my culminating project for the gr. 12 Computer Science class I took online.
