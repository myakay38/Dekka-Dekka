Introduction

Dekka Dekka is a three-dimensional maze game with a two-dimensional display. A grid of boxes displaying images shows the maze to the user. Each box contains either a background image of empty space, or an image of an asteroid as an obstacle. The avatar, which is an alien in a rocketship, can be moved with the arrow keys to navigate through the “empty” boxes, but not through obstacles. However, some obstacles are not so difficult to overcome if you only change your perspective. Buttons beside the grid allow the user to “rotate” the display to view the maze from above, below, behind, in front, and both sides. From these different angles, the user will find ways to get around the obstacles and reach the alien’s home planet at the end of the maze.
